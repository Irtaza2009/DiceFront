# DiceFront

A submission for [milkyway](https://milkyway.hackclub.com/)!!

**Play at:** https://irtaza.itch.io/dicefront

## Description
**DiceFront** is a 2-Player **turn-based strategy** game where luck meets logic.

**Roll dice**, **conquer territories**, and **outplay** your opponent in fast, tactical battles where every attack is a risk, and every roll can change the front line.

You can play against **AI**, or against a **friend**!!

## Controls

### Mouse Only

- Click a territory to select it

- Click an adjacent enemy territory to attack

- End Turn to let the AI make its move

|      Input       |       Action     |
|------------------|------------------|
|    Left Click    | Select Territory |
|    Left Click    | Attack Territory |
| End Turn Button  |     End Turn     |
|   Pause Button   |    Pause Menu    |
